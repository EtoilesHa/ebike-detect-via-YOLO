
# ebike-detect-via-YOLO

### 内容简介： 
- 分别基于YOLO_v5与优化的DAMO-YOLO算法，通过数据增强，模型训练，模型迁移与应用，对电梯场景中的人与电动车进行识别，实现电动车进入电梯的预警功能。
